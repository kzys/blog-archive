Focus

2010/12/21 22:04

steps to phantasien にはやたら影響を受けていて、5月から4ヶ月間とすこし自宅のインターネット環境を貧弱にしていたのもそのひとつ。職場の人に「自宅からは iPhone くらいでしか見れなくて」なんていって不思議がられたけど、そういう意識ですごしている本人は

  • Paul Graham の Disconnecting Distraction とか (いまみたら、うまくいかなかったって注意書きがついてた...)
  • Obama 大統領の Hampton University でのスピーチとか (The Economist は Barack Obama's rant against technology と冷ややか)
  • また The Economist に Stay on target という記事があったとか
  • BBC に How to disconnect from your online life という記事 (これはプライバシーより) があったとか
  • Joel Spolsky は Twitter やらない とか (いまみたら、やってた...)
  • iDevBlogADay でたまたま Focus というのを読んだとか
  • "simplicity in the age of distractions" をうたう Focus という本がでたとか

そういう情報ばかりが目にとまるようになるので、そこまで変なことをしている気にはならなかった。

Twitter や Facebook は、見るたびに新しい情報があって、行動に対してすぐさま反応がかえってくる「速い」メディアだと思う。それが楽しくてのめりこみやすい、というのはわかる。ただ、それにのめりこむのが自分にとって良いことなのかというと、よくわからない。

「X と Y は提供する価値がちがうので食い合わない」みたいなことをいうひとがいる。ある程度のサイズの人の集合をとればそうなんだろうけど、1人の1日は24時間しかなく、自分にとってはかなりが食い合っている。それで、24時間のなかの楽しみの時間を奪い合うのはもういいかなあと、最近すこし思う。

Mobile First Web Second

2010/12/01 23:43

oogatta さんが

「モバイル（この場合スマートフォン）の方が作り手もユーザ視点になりやすい」って言ってたけどその通りだなあ。なんでしょうね、なんでもできてしまうとどんどんユーザの視点から離れる…。って悲しいですね。

と Twitter に書いていて、誰それ? と聞いたら Mobile First Web Second (continued) とかの覚え違いかも、というので読んでみた。

"continued" というからにはその前があり、9月の Mobile First Web Second は

  • Twitter とかモバイルから使われる方が多いらしいですね
  • Foursquare はもっとで、というかモバイルじゃないと意味ないよね
  • 空き時間とかに使えていいよね

とかそういう感じだった。"But because they are also web apps and connected to a web scale network, they can offer a lot of value that mobile only apps cannot." というのはよくわからず。web scale network につながらない mobile only apps ってなんなんだろう。

11月の Mobile First Web Second (continued) はもっと踏み込んでいて

  • 最近 (たぶん 3月) の AVC の見た目の変更は、ようするにモバイルの Web サイトっぽくしたかった。iPad で見る Web が好きなので、AVC も iPad でかっこよくなるようにしたかった。
  • じぶんのみてる (Fred Willson は VC なので) 会社の新しい Web サービスのデザインがぐちゃぐちゃしててダメで、その会社のリリース予定の iPhone アプリをみて「Web もこうすればいいじゃん」といった。
  • モバイル Web を制約として Web デザインについて考えるのは、これから大きくなってくるだろうね。

という感じ。モバイルの画面の狭さとか回線の遅さとかを仮定してみると、全体のなかで落とすべき部分とそうじゃない部分の判断がつくよ、とかそういう話だと思う。また "you are likely to keep them on the mobile app and off the web app." がよくわからない…

最近の Web

2010/11/08 23:57

最近ひさびさに新しくアカウントをとったのが Instagram で

  • iPhone で写真をとって
  • トイカメラとかそういう風の加工をして
  • アップロードして
  • フォローしたりされたりして
  • フォローした人々の新しい写真が一覧できて
  • コメントつけたりライクしたりできて

と書き出すとぜんぜんだめな感じがするんだけど、使ってみるとそうでもなくて、ぽつぽつ写真をアップロードしている。

ちゃんと写ってる写真を白く飛ばしたりとか、周辺減光させてみたりとか、前はすごくかっこわるいと思ってたんだけど、だんだん慣れてきた。ただ (ピチカート・ファイヴの「大都会交響楽」のビデオみたいな) フィルムっぽい枠をつける加工はさすがに嘘すぎて使えていない。あと食事の写真をとるのも苦手で、生活の写真をとろうと思うと、自分の生活を「良く」見せようとする作為に自分で気づいて、恥ずかしくなって途中でやめてしまう。

Facebook

Facebook も見てはいる。流れのはやいスレ (?) に書き込むと、ページ遷移する間もなくいろいろ画面上に通知がとどいたりして、新しい世代のソフトウェアをさわっている気持ちになる。すごい。

ただ、通知と一緒にとどくメールは送りすぎだと思う。なんとかしようと設定画面をみたら、チェックボックスが50個くらいならんでいて、かっこわるかった。

結局、Facebook はソーシャルの Microsoft Office みたいなもので、世界は征服して、大学を中退したひとは大金持ちになるかもしれないけど、かっこよくはないなあと思う。あるいはこういう設定地獄みたいなものがメジャー感で、メジャー感をかっこよく思えない自分がだめなのかもしれない。
